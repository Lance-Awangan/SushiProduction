# Ingredient Rules

## Overview

This document defines the ingredient breakdown for each sushi product.

Each product rule describes how many slices or portions of each ingredient are required per pack.

Source: legacy PWA `DEFAULT_PRODUCT_NETA_MAP`.

---

## Data Structure

```dart
class IngredientRule {
  final String productName;
  final Map<String, int> ingredients;
}
```
